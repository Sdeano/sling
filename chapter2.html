Adding Approve and Reject Actions

Step 2 - Initiate Swipe for Approval

Especially on iOS, users are used to trigger default actions on lists using swipe actions instead of navi-gating to a detail view. This should also be possible in our approval application. So we need to implement a swipe event on the list. In in view/Master.view.xml perform:


<List
d="list"
[…]
updateFinished="onUpdateFinished"
swipe="onSwipe"
selectionChange="onSelectionChange">
[…]
<swipeContent>
<Button id="swipeButton" press="onSwipeApprove" text="{i18n>xbut.approve}" type="Accept"/>
</swipeContent>
</List>
